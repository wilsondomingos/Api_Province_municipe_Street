

## About Web service Province municipe and street
Este web service foi desenvolvido por mim Wilson Mateus Domingos, com objectivo de ajudar desenvolvedor com sistemas em necessitam de dados de provincia, municipio, destritos, bairros e comunas da Angola.

## Learning Web service
Este Web service é composto por três metodos, um que fornece todas as provincias, outro todos os municipio de uma determinada provincia, e o ultimo metodo fornece todos os destritos ou comuna de uma determinada provincia.
